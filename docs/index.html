Microsoft GitHub Repo Maker

This app allows a user to make an existing project into a GitHub repo

Instructions for using this program:

Step 1

Click 'Open Directory Explorer' and pick a project from the files on your computer that you want to make into a GitHub repo

Step 2

Enter what you want the GitHub repo name to be

Step 3

Enter what you want the GitHub repo description to be

Step 4

Choose if you want the repo to be public or private

Step 5

Enter your GitHub username

Step 6

Enter your GitHub Access Key

Step 7

Click on 'Submit Repo Info' to create the GitHub repo
